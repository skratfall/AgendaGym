# 🎬 Guía Rápida - Sistema de Filtros Admin

## Acceso a los Filtros

### En la pantalla de "Todas las Citas":
1. Toca el **icono de filtro** (⊙) en la barra superior derecha
2. El **panel de filtros** se abrirá

## Opciones de Filtro Disponibles

| Filtro | Descripción | Uso |
|--------|-------------|-----|
| **Usuario** | Mostrar citas de un usuario específico | Selecciona de la lista de usuarios |
| **Servicio** | Mostrar citas de un tipo de servicio | Yoga, CrossFit, Boxeo, etc. |
| **Estado** | Mostrar citas por estado | PENDING, CONFIRMED, COMPLETED, CANCELLED |
| **Rango de Fechas** | Mostrar citas dentro de un período | Desde-Hasta |

## Ejemplos de Uso

### Caso 1: Ver todas las citas de Yoga
1. Abre filtros (⊙)
2. En "Servicio" selecciona "Yoga"
3. Se filtran automáticamente

### Caso 2: Ver citas de un usuario específico en una fecha
1. Abre filtros (⊙)
2. Selecciona usuario en "Usuario"
3. Selecciona fecha en "Rango de fechas"
4. Se muestran solo esas citas

### Caso 3: Ver citas pendientes de CrossFit
1. Abre filtros (⊙)
2. Selecciona "CrossFit" en Servicio
3. Selecciona "PENDING" en Estado
4. Se filtran en tiempo real

## Limpieza de Filtros

- Toca el botón **"Limpiar"** en el panel de filtros (cuando hay filtros activos)
- O selecciona "Todos los usuarios", "Todos los servicios", etc.

## Indicadores Visuales

- 🔵 Cuando hay **filtros activos**, el panel muestra valores específicos
- 📋 Cuando no hay citas, se muestra "No hay citas disponibles"
- ✨ Los filtros se aplican **en tiempo real** mientras escribes o seleccionas

## Combinaciones Potentes

✨ **Puedes usar múltiples filtros simultáneamente:**
- Usuario: "María" + Servicio: "Yoga" + Estado: "CONFIRMED"
- Servicio: "CrossFit" + Rango de fechas: 01/06/2025 - 07/06/2025

## Notas
- Los filtros **se reestablecen** al salir y volver a entrar a la pantalla
- Puedes **editar o eliminar citas** directamente desde la vista filtrada
- Los cambios de estado se aplican **sin salir** de los filtros
